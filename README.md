# Project 3 - The Visual Identity 

## DecodeLabs UI/UX Internship

### Overview

This project focused on transforming low-fidelity wireframes into a high-fidelity user interface.

The goal was to establish a complete visual identity through colors, typography, components, and layouts.

---

## Scenario

A premium organic food delivery service wanted a modern visual identity that communicates trust, freshness, health, and speed.

---

## My Approach

I created a fictional organic food delivery brand called **Earthly**.

### Style Tile

I developed:

* Color Palette
* Typography System
* Buttons
* Product Card Component

### Brand Identity

Earthly represents the journey of fresh organic produce from farms directly to homes.

### Screens Designed

* Splash Screen
* Product List Screen
* Product Details Screen

---

## Deliverables

* Style Tile
* Design Components
* High-Fidelity UI Screens

---

## Tools Used

* Figma

---

## Key Learning

This project taught me how colors, typography, spacing, and consistency work together to create a trustworthy user experience.
